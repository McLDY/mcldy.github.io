Blog使用的音乐Api

🎵 Music API 文档（基于 Cloudflare Worker）

本 API 提供统一的音乐数据访问接口，支持多平台（当前为 bilibili 与 netease）。
所有接口均返回 JSON，且支持跨域（Access-Control-Allow-Origin: *）。

📌 基础信息

  • Base URL：https://mbma.my20130910.workers.dev
  • 响应格式：application/json; charset=utf-8
  • 跨域支持：是
  • 错误格式统一：

📚 API 列表

方法 路径 描述
GET /search 搜索歌曲/视频
GET /song 获取歌曲播放信息（APlayer 友好）
GET /lyric 获取歌词

🎯 1. 搜索接口 /search

请求方式

请求参数

参数 类型 必填 默认值 说明
keyword string 是 无 搜索关键词
page number 否 1 页码（从 1 开始）
source string 否 bilibili 数据源：bilibili 或 netease

成功返回示例

⚠ 注意：不同平台返回字段可能略有差异，但 Worker 不做二次加工，直接返回插件结果。

错误返回

🎯 2. 获取歌曲 /song

请求方式

请求参数

参数 类型 必填 默认值 说明
id string 是 无 歌曲 ID（平台原始 ID）
quality string 否 standard 音质（由插件决定是否支持）
source string 否 bilibili 数据源：bilibili 或 netease

成功返回示例（APlayer 友好）

⚠ 注意：

  • B 站返回的 url 可能是 .m4s（DASH），浏览器无法直接播放。
  • 网易云外链可能不稳定（偶尔 404 / HTML）。
  • Worker 不做任何代理或修复，只返回插件结果。
